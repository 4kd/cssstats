# Rprtr

## Global Dependencies

Install [node.js](http://nodejs.org) and then open Terminal:

    npm install -g coffee-script
    npm install -g grunt-cli
    npm install -g node-dev

## Project Setup

    git clone git@github.com:mrmrs/rprtr.git
    cd rprtr
    npm install
    grunt build:debug
    node-dev app.coffee

## Heroku

    ./heroku.sh
