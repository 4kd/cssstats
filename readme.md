# [:bar_chart:](https://cssstats.com) Css Stats

Potentially interesting stats on stylesheets.

## Installation

```
git clone https://github.com/cssstats/cssstats
cd cssstats
yarn
```

## Usage


#### Development

```
yarn dev
open http://localhost:3000
```

#### Production

```
yarn build
yarn start
```
